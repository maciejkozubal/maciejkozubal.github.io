## Running Locally

To run the site locally:

1. **Install Dependencies**:

   Ensure you have Ruby and Bundler installed. Then, install the required gems:

   ```bash
   bundle install
    ```
2. **Serve**

    Start the Jekyll server:

    ```bash
    bundle exec jekyll clean
    bundle exec jekyll serve
    ```
